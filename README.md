# Inventory Management System 🧾📦

A full-stack web app using:

- ⚛️ React + TypeScript (frontend)
- 🐍 .NET 8 Web API + Entity Framework (backend)
- 🗃️ SQL Server

## Features
- Add products
- View inventory
- Real-time sync
- CORS enabled

## Getting Started

```bash
cd backend/InventoryApi
dotnet run

cd frontend/inventory-ui
npm install
npm run dev
